# astroTools
Outis informatiques pour le traitement d'images astro

## Dependances
Dépendances nécessaires à la compilation:
- opencv-2.4.13
- tiff-3.8.0
- LibRaw-0.18.2
- jpeg-6b
- bitmap

## Compilation
Après avoir compiler les dependances, renseigner leur chemin dans le config.sh
Il est nécessaire de sourcer le script "config.sh" avant chaque compilation ou utilisation par la commande suivante:

. config.sh
